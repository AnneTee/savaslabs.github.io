This is the site for [Savas Labs](http://savaslabs.com).

The site is built using Jekyll. To run locally:

1. Clone the repo
2. `bundle install`
3. `bundle exec jekyll serve --config _config.yml,_config.test.yml`

To run the tests:

`$ bash _tests/run-tests.sh`
